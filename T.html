It is not a very linear relation, but we have to start with something anyway :)¶

$$ Loss(w, b) = \frac{1}{n}\sum_{i=1}^{n}[\hat{y}_i - y_i]^2 \\ \\ \hat{y} = wx + b $$

The basic algorithm is¶

$$ \text{repeat until Loss(w, b) is small enough} \\ w = w - lr * \frac{\partial{Loss(w, b)}}{\partial{w}} \\ b = b - lr * \frac{\partial{Loss(w, b)}}{\partial{b}} $$

And do not worry too much about the calculus¶

$$ \frac{\partial{Loss(w, b)}}{\partial{w}} = \frac{2}{n}\sum_{i=1}^{n}[(wx_i+b) - y_i] x_i\\ \frac{\partial{Loss(w, b)}}{\partial{b}} = \frac{2}{n}\sum_{i=1}^{n}[(wx_i+b) - y_i] $$
In [310]:
def predict(X, w, b):
    return np.dot(w, X) + b

def mean_squared_error(X, w, b, y):
    y_hat = predict(X, w, b)
    n = len(y)
    
    return (1/n) * np.sum(np.square(y - y_hat))

def partial_derivative(X, w, b, y):
    y_hat = predict(X, w, b)
    n = len(y)
    dB = (2/n) * np.sum((y_hat - y))
    dW = (2/n) * np.dot((y_hat - y), X)
    return dB, dW

def linear_regression(X, w, b, y, lr=0.0001, steps=1000, info_each=100):
    
    for step in range(0, steps):
        error = mean_squared_error(X, w, b, y)
        if step % info_each == 0:
            print("Iteration %d error %.3f" % (step, error))
        
        dB, dW = partial_derivative(X, w, b, y)
        w = w - lr * dW
        b = b - lr * dB
    print("Final error %.3f" % error)
    return w, b

A single variable gives us an error of ~140829942 notice that this is the squared error, we are going to use a slightly different metric, the mean absolute error.¶

$$ MAE = \frac{1}{n} \sum_{i=1}^{n}|\hat{y}_i - y_i| $$

We do not use MAE to perform any machine learning calculations, it is just a way to now better how good the prediction is, so lets modify slightly our linear_regression algoritithm¶

    n = len(y)
    return (1/n) * np.sum(np.absolute(y - y_hat))

The next thing we want to look is normalization, if you check the columns on X, you will notice that their magnitudes are very different (ie: number of children is normally 0 to 3 while bmi goes from 16-30.¶

We want all the columns to be around 0, so we get numerical stability.¶

A simple way to do this is with max-min normalization¶

$$ X_{normalized} = \frac{X - min(X)}{max(X) - min(X)} $$
In [336]:
def normalize(X):
    X_normalized = pd.DataFrame()
    for column in X.columns:
        X_normalized[column] = (X[column] - min(X[column])) / (max(X[column]) - min(X[column]))
    return X_normalized
